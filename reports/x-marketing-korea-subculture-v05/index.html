질문답
X를 한국 대중 도달 플랫폼으로 봐야 하는가아니오. YouTube 대비 도달 규모가 작고, Instagram/TikTok 대비 이용률 성장도 정체권이다.
X를 서브컬처 팬덤 운영 채널로 봐야 하는가예. 공식 정보, 팬아트, 실시간 반응, 커뮤니티 재확산이 연결된다.

실무 권장안YouTube는 영상 허브, X는 네이티브 세로 숏폼과 팬덤 반응 증폭, 커뮤니티는 확산 검증 레이어로 분리 운영한다.

5.1 알려진 시간 경과 패턴

알려진 사실실무 해석보고서 적용
공개 tweet impression은 평균적으로 게시 후 약 72초 지점에서 초당 impression 피크에 도달첫 수분의 초기 반응 속도가 배포 폭을 크게 좌우업로드 직후 15분/1시간 캡처를 별도 보관
tweet impression의 중앙 half-life는 약 80분일반 organic 게시물은 1~2시간 안에 1차 확산력이 크게 결정1시간 reach와 6시간 reach를 early velocity 지표로 사용
약 95%의 tweet은 24시간 뒤 유의미한 신규 impression이 거의 없음24시간 수치는 대다수 organic 게시물의 1차 성과 판단점24h 스냅샷을 캠페인 비교 기준으로 고정
Brandwatch 기준 X 콘텐츠 지표는 게시 15분 후 첫 refresh, 이후 30일간 최소 일 1회 refresh단일 시점 캡처만으로는 초반 기울기와 long-tail을 분리하기 어려움15m/1h/6h/24h/72h/7d/30d 캡처 템플릿 필요

주의: 위 연구는 X/Twitter 게시물 impression 일반 패턴이다. X video view 정의와 완전히 같지는 않으므로, 여기서는 “동영상 reach가 시간에 따라 어떻게 누적/감쇠되는가”를 판단하는 보조 프레임으로 사용한다.

5.7 링크 클릭률과 시청 품질 지표

X 공개 게시물에서 보이는 view/reach는 링크 클릭률이나 평균 시청시간을 직접 보여주지 않는다. 따라서 공개 리포트에서는 “알려진 광고 benchmark”와 “공식 게임 계정 organic 게시물에서 확인 가능한 범위”를 분리해야 한다.

지표알려진 공개 정보이번 리포트 적용
X 광고 CTR공개 benchmark는 대체로 약 0.86%~2% 범위로 언급된다. 일부 자료는 promoted post 평균 CTR 0.86%, 일반 X 광고 CTR 1~2% 수준을 제시한다.paid/boosted 소재의 참고 하한선으로만 사용
게임 공식 계정 organic 링크 클릭률공신력 있는 공개 평균 benchmark는 확인하기 어렵다. X 공개 UI에도 link click은 표시되지 않는다.관리자 analytics, UTM, shortlink 로그 없이는 계산 불가
YouTube 링크형 게시물의 공개 대체 지표공개 UI에서는 view/reach, repost, reply, like, bookmark 정도만 확인 가능하다.link click 대신 X/YT 배율과 repost/bookmark 밀도를 보조 지표로 사용
X 직접 업로드형 시청 품질평균 시청시간, 25/50/75/100% completion rate는 공개 게시물 캡처로 확인 불가X analytics 또는 Brandwatch/Sprinklr 등 측정 도구 필요
운영 질문현재 답
X에 YouTube 링크를 올리면 클릭률이 몇 %인가공개 데이터만으로는 알 수 없다. 광고 benchmark 0.86~2%는 paid 참고치일 뿐 organic 공식 계정에 직접 적용하면 위험하다.
X view가 높으면 YouTube 유입도 높다고 볼 수 있는가아니다. X view는 타임라인 도달/노출성 지표이고, YouTube 유입은 link click/UTM으로 별도 확인해야 한다.
이번 데이터로 볼 수 있는 것X에 미러링했을 때 YouTube 조회수 대비 몇 배의 공개 reach가 발생하는지, 그리고 repost/bookmark 등 팬덤 반응 밀도
추가 확보하면 좋은 것게시물별 link click, detail expand, profile click, video completion, 평균 시청시간, UTM 유입 세션

5.10 해석

관찰판단
블루 아카이브 8건 중 7건은 X view가 YouTube 조회수보다 큼X는 서브컬처 공식 영상의 보조 링크 채널이 아니라 독립 reach 채널로 작동 가능
레이사 매지컬은 X/YT 32.04x로 가장 큼캐릭터 중심 세로 네이티브 영상은 X에서 YouTube 대비 압도적으로 튈 수 있음
유형별 평균은 세로 직접 업로드 8.93x, 가로 링크형 2.42xYouTube 조회수 기준 X reach 환산 시 유형별 계수를 분리해야 함
가로 YouTube 링크형도 1.96x~2.88x링크형은 공지/유입 목적에 유효하지만 네이티브 세로형과 역할을 분리해야 함
링크 클릭률은 공개 UI로 확인되지 않음X view가 높아도 YouTube 유입/전환 기여는 link click·UTM 없이 단정하지 않음
니케는 YouTube 조회수가 X view보다 약 4배 큼음악·스테이지형 콘텐츠는 YouTube 소비가 더 강하게 작동한 사례
공개 참여율은 1.18%~7.27%로 편차가 큼view 규모와 참여 밀도는 별개로 봐야 함
quote 수와 link click은 캡처상 미확인관리자 지표 또는 추가 캡처가 있으면 투자 판단 정확도 상승
