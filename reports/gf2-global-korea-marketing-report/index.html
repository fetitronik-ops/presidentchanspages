6. 종합 관찰

소녀전선2의 마케팅은 글로벌 공식 YouTube의 운영형 영상 콘텐츠와 지역별 오프라인 접점이 결합된 형태다. 글로벌 채널은 캐릭터 PV와 패치 프리뷰를 중심으로 업데이트를 콘텐츠화하고, 일본·미국은 팝업/상권 이벤트, 중국은 지도앱 같은 생활 서비스 콜라보, 한국은 카페·전국 제휴·AGF·갤러리·팬 소통 행사처럼 팬덤 밀착형 활동이 두드러진다.

한국 사례를 포함하면 소녀전선2의 마케팅은 단순 온라인 광고보다 팬이 직접 방문하고 인증하고 굿즈를 구매하며 커뮤니티에 공유할 수 있는 경험형 마케팅에 강점이 있음을 확인할 수 있다.

주요 출처: 공식 YouTube @GFL2EXILIUM 브라우저 확인, Sunborn/GF2 공식 페이지 검색 결과, Essential Japan, The KON Network, Reddit r/GirlsFrontline2·r/gachagaming 공개 게시물, IOP Wiki, Big Games Machine 론칭 PR 자료, GameY·Ruliweb·국내 게임 매체의 한국 오프라인 이벤트 보도. 외부 출처는 공개 웹 자료 기준이며 일부 이벤트 세부사항은 커뮤니티/보도자료 기반입니다.
